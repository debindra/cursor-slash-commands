# Improve Agent Workflow

Optimize and improve AI agent prompts and performance through iterative refinement.

## Usage
/improve-agent [agent description or prompt to improve]

## Example
/improve-agent Optimize the code review agent to catch more security issues

## Workflow Steps

1. **Analysis Agent**
   - Analyze current agent prompt/behavior
   - Identify weaknesses and gaps
   - Review performance metrics
   - Compare with best practices
   - Identify improvement opportunities

2. **Optimization Agent**
   - Refine prompt engineering
   - Add context and examples
   - Improve instruction clarity
   - Add error handling guidance
   - Optimize for specific use cases

3. **Testing Agent**
   - Test improved agent on sample tasks
   - Compare before/after results
   - Measure improvement metrics
   - Validate edge cases
   - Check for regressions

4. **Documentation Agent**
   - Document improvements made
   - Create usage examples
   - Update agent description
   - Add best practices notes

## Output
- Improved agent prompt/configuration
- Performance comparison report
- Test results
- Documentation updates
- Usage guidelines

## Variables
$ARGUMENTS - Agent description or prompt to improve

## Optimization Areas
- Prompt clarity and specificity
- Context management
- Error handling
- Output formatting
- Performance tuning
- Edge case handling
