# Bundle Analyze Tool

Analyze and optimize JavaScript bundle sizes with detailed breakdowns.

## Usage
/bundle-analyze [build output or "current"]

## Example
/bundle-analyze dist/

## Analysis Features

1. **Bundle Breakdown**
   - File size analysis
   - Module size breakdown
   - Dependency tree visualization
   - Duplicate detection
   - Unused code detection

2. **Optimization Suggestions**
   - Code splitting opportunities
   - Tree shaking potential
   - Dynamic import suggestions
   - Dependency optimization
   - Compression opportunities

3. **Comparison**
   - Before/after comparisons
   - Version comparisons
   - Build comparison
   - Performance impact

4. **Visualization**
   - Bundle size charts
   - Dependency graphs
   - Size trends
   - Interactive reports

## Output
- Bundle analysis report
- Optimization recommendations
- Visual reports
- Action items

## Variables
$ARGUMENTS - Build output directory or "current"

## Supported Tools
- webpack-bundle-analyzer
- rollup-plugin-visualizer
- vite-bundle-visualizer
- parcel-bundler analyzer

## Optimization Strategies
- Code splitting
- Tree shaking
- Dynamic imports
- Lazy loading
- Compression
- Minification
