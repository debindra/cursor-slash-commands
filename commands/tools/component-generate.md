# Component Generate Tool

Generate UI components with best practices and accessibility.

## Usage
/component-generate [component name] [type]

## Example
/component-generate UserCard functional with TypeScript

## Component Types

1. **React Components**
   - Functional components
   - Class components
   - Hooks integration
   - TypeScript support

2. **Vue Components**
   - Options API
   - Composition API
   - TypeScript support

3. **Angular Components**
   - Component with template
   - TypeScript
   - Dependency injection

4. **Styling Options**
   - CSS Modules
   - Styled Components
   - Tailwind CSS
   - SCSS/SASS

## Output
- Component file
- Styles file
- Test file
- Storybook story (if applicable)
- Documentation

## Variables
$ARGUMENTS - Component name and type

- TypeScript types
- PropTypes (if applicable)
