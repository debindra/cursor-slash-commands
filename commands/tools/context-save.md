# Context Save Tool

Save current project context for later restoration across sessions.

## Usage
/context-save [context name] [description]

## Example
/context-save authentication-feature "User auth implementation in progress"

## Features

1. **Context Capture**
   - Current file states
   - Open files
   - Project structure
   - Recent changes
   - Configuration state

2. **Metadata**
   - Context name
   - Description
   - Timestamp
   - Tags
   - Related files

3. **Storage**
   - Save to context file
   - Version control
   - Searchable index
   - Quick access

## Output
- Context snapshot file
- Metadata file
- Index entry

## Variables
$ARGUMENTS - Context name and description

## Use Cases
- Save work in progress
- Context switching
- Multi-session projects
- Team context sharing
- Project milestones
