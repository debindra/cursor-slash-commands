# Cost Optimize Tool

Analyze and optimize cloud infrastructure costs with actionable recommendations.

## Usage
/cost-optimize [infrastructure type] [requirements]

## Example
/cost-optimize AWS infrastructure with EC2 and RDS

## Analysis Types

1. **Resource Analysis**
   - Identify underutilized resources
   - Find over-provisioned instances
   - Analyze storage costs
   - Review network costs
   - Check data transfer costs

2. **Optimization Strategies**
   - Right-sizing recommendations
   - Reserved instance planning
   - Spot instance opportunities
   - Auto-scaling optimization
   - Storage tier optimization

3. **Cost Allocation**
   - Cost by service breakdown
   - Cost by environment
   - Cost by team/project
   - Cost trends analysis

4. **Recommendations**
   - Immediate cost savings
   - Long-term optimization
   - Reserved capacity planning
   - Architecture improvements

## Output
- Cost analysis report
- Optimization recommendations
- Resource right-sizing plan
- Cost allocation breakdown
- Implementation guide

## Variables
$ARGUMENTS - Infrastructure type and requirements

## Supported Cloud Providers
- AWS (EC2, RDS, S3, Lambda, etc.)
- Azure (VMs, SQL Database, Blob Storage, etc.)
- GCP (Compute Engine, Cloud SQL, Cloud Storage, etc.)

## Optimization Areas
- Compute resources
- Storage optimization
- Network costs
- Database optimization
- Serverless optimization
- Reserved capacity
