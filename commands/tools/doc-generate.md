# Documentation Generate Tool

Generate comprehensive documentation from code with multiple formats.

## Usage
/doc-generate [target: file, directory, or "project"]

## Example
/doc-generate src/api/

## Documentation Types

1. **API Documentation**
   - OpenAPI/Swagger specification
   - Endpoint descriptions
   - Request/response schemas
   - Authentication requirements
   - Example requests

2. **Code Documentation**
   - JSDoc/TSDoc comments
   - Function descriptions
   - Parameter documentation
   - Return type documentation
   - Usage examples

3. **Component Documentation**
   - React/Vue component docs
   - Props documentation
   - Usage examples
   - Storybook stories

4. **Architecture Documentation**
   - System architecture diagrams
   - Component relationships
   - Data flow diagrams
   - Database schema docs

## Output
- API documentation (OpenAPI/Swagger)
- Code documentation files
- README updates
- Architecture diagrams
- Usage examples

## Variables
$ARGUMENTS - Target file, directory, or "project"

## Formats Supported
- Markdown
- HTML
- PDF
- OpenAPI/Swagger (JSON/YAML)
- JSDoc/TSDoc

## Documentation Standards
- OpenAPI 3.0
- JSDoc standards
- TSDoc standards
- Markdown best practices
- README templates
