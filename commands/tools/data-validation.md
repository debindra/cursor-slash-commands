# Data Validation Tool

Generate data validation schemas and quality checks.

## Usage
/data-validation [data type] [validation requirements]

## Example
/data-validation user registration data with email and phone validation

## Validation Types

1. **Schema Validation**
   - Data type validation
   - Required field checks
   - Format validation
   - Range validation
   - Enum validation

2. **Business Rules**
   - Custom validation rules
   - Cross-field validation
   - Conditional validation
   - Business logic checks

3. **Data Quality**
   - Completeness checks
   - Uniqueness validation
   - Consistency checks
   - Accuracy validation
   - Timeliness checks

4. **Integration**
   - API validation
   - Database constraints
   - Form validation
   - File validation

## Output
- Validation schemas
- Validation functions
- Error handling
- Test cases
- Documentation

## Variables
$ARGUMENTS - Data type and validation requirements

## Supported Frameworks
- JavaScript: Joi, Yup, Zod
- Python: Pydantic, Cerberus, Great Expectations
- Java: Bean Validation (JSR 303)
- C#: Data Annotations, FluentValidation
- Go: go-playground/validator

## Validation Patterns
- Schema-based validation
- Rule-based validation
- Custom validators
- Async validation
- Conditional validation
