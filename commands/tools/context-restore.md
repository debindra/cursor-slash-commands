# Context Restore Tool

Restore previously saved project context.

## Usage
/context-restore [context name or "list"]

## Example
/context-restore authentication-feature

## Features

1. **Context Retrieval**
   - List available contexts
   - Search contexts
   - View context details
   - Restore context

2. **Restoration**
   - Restore file states
   - Open relevant files
   - Restore configuration
   - Resume work state

3. **Management**
   - View context history
   - Delete old contexts
   - Update context metadata
   - Merge contexts

## Output
- Restored context
- Opened files
- Restored state
- Context information

## Variables
$ARGUMENTS - Context name or "list"

## Restoration Options
- Full restore
- Selective restore
- Merge with current
- Preview before restore
