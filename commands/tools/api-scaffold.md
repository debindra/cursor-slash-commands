# API Scaffold Tool

Generate RESTful API endpoints with best practices and complete implementation.

## Usage
/api-scaffold [resource] [operations]

## Example
/api-scaffold products CRUD with pagination and filtering

## Features
- Generate route handlers
- Create controllers with business logic
- Add input validation schemas
- Implement error handling
- Add authentication/authorization middleware
- Generate database models
- Create migration files
- Add API documentation

## Output
- Route files
- Controller files
- Model files
- Validation schemas
- Error handlers
- API documentation

## Variables
$ARGUMENTS - Resource name and operations

## Supported Frameworks
- Express.js (Node.js)
- FastAPI (Python)
- Spring Boot (Java)
- ASP.NET Core (C#)
- Gin (Go)
- Rails (Ruby)

## Operations Supported
- CRUD (Create, Read, Update, Delete)
- Pagination
- Filtering
- Sorting
- Search
- Bulk operations
